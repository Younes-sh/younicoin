# younicoin 

### Technologies used :

- React.js
- Node.js

#### React.js 
- `npm instal `
- `npx-react-app coin`
### Node.js
- `npm install node`
- `npm install express`

### Dependencies :

- lightweight-charts for chart crypto   `npm i lightweight-charts`
- mui/material   `npm i @mui/material`
- website -> [younicoin](https://younicoin.netlify.app/) 

![](./coin/src/asset/chart.gif)

npm run build
